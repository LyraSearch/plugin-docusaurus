# Docusaurus Lyra Search Plugin

## Usage

Install the plugin:

```bash
npm install --save @lyrasearch/plugin-docusaurus
```

```bash
yarn add @lyrasearch/plugin-docusaurus
```

Add the plugin to your `docusaurus.config.js`:

```js
plugins: ["@lyrasearch/plugin-docusaurus"];
```
